## Behavior

### Vocabulary

| Internal State | UI Label | Color | Meaning |
|----------------|----------|-------|---------|
| High co-activation count | `recently reinforced` | green | Seen a lot lately |
| Conflicting evidence | `contested` | amber | Sources disagree |
| Low recent access | `decaying` | gray | Not touched recently |
| High salience rank | `high-traffic path` | blue | Load-bearing node |
| Derived from corrections | `shaped by corrections` | teal | Human review updated it |
| Bridge edge rescue | `cross-domain link` | purple | Connects clusters |
| Projection stale | `memory out of sync` | red | Graph lagging |

### Component API

```tsx
interface MemorySignalChipProps {
  signal: SignalType;
  count?: number;  // for "reinforced ×N"
  timestamp?: string;  // for "N days ago"
  onClick?: () => void;
}

type SignalType =
  | 'recently-reinforced'
  | 'contested'
  | 'decaying'
  | 'high-traffic'
  | 'shaped-by-corrections'
  | 'cross-domain'
  | 'out-of-sync';
```

### Display

```
[recently reinforced]     [contested]     [decaying]
[high-traffic path]       [shaped by corrections]
[cross-domain link]       [memory out of sync]
```
