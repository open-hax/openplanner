## Purpose

Propagate review decisions to the memory graph as signals.

---

## Scope

Wire review actions to graph memory signal emission.

---

## Behavior

### Signal Types

| Review Action | Memory Signal | Effect |
|---------------|---------------|--------|
| Approve | reinforcement | Increases node strength |
| Reject with correction | contradiction | Marks node contested, adds correction |
| Edit + approve | correction | Updates node content, marks "shaped by corrections" |

### Write-Back Flow

```
User approves item
    │
    ├─ 1. Get node(s) associated with item
    ├─ 2. Emit reinforcement signal to graph memory
    ├─ 3. Update node strength
    └─ 4. Log signal in audit trail
```

### Correction Flow

```
User rejects with notes
    │
    ├─ 1. Get node(s) associated with item
    ├─ 2. Emit contradiction signal
    ├─ 3. Mark node as "contested"
    ├─ 4. Store correction text
    └─ 5. Log signal in audit trail
```

### Visibility

- Written signals appear in Memory Inspector
- Memory Activity Feed (1.3) shows recent corrections
- Node badges update with signal chips

---

## Exit Criteria

- [ ] Approve → reinforcement signal emitted
- [ ] Reject → contradiction signal emitted
- [ ] Edit → correction signal emitted
- [ ] Signal visible in Memory Inspector
- [ ] Signal appears in Dashboard activity feed
- [ ] Audit trail records all signals

---

## Out of Scope

- Signal strength tuning
- Custom signal types
- Signal reversion
