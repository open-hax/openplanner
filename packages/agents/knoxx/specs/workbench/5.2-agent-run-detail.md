## Behavior

### Detail Layout

```
┌─────────────────────────────────────────────────────────────────┐
│ Run: synthesis-q4-summary                                       │
│ ─────────────────────────────                                   │
│ Goal: synthesize Q4 KPIs from devel-docs lake                   │
│ Scope: collection=devel-docs, date≥2025-10-01                   │
│ Tools: query/answer, cms/documents                              │
│ Status: awaiting human approval                                 │
│ Confidence: 0.71                                                 │
│ Step 3/4: output generated                                      │
│                                                                 │
│ ─────────────────────────────────────────────────────────────── │
│ Step Trace:                                                     │
│ 1. ✓ Query devel-docs for Q4 content                           │
│ 2. ✓ Extract KPI mentions                                      │
│ 3. ✓ Generate summary                                          │
│ 4. ⏸ Awaiting approval                                         │
│                                                                 │
│ [Approve]  [Revise Goal]  [Stop]  [View Output]                │
└─────────────────────────────────────────────────────────────────┘
```

### Run Information

| Field | Content |
|-------|---------|
| Goal | What the agent is trying to accomplish |
| Scope | Collection/filters constraining the run |
| Tools | Which tools the agent has used |
| Status | Current state |
| Confidence | Model confidence (0.0-1.0) |
| Progress | Step X/Y |

### Control Actions

| Action | Effect |
|--------|--------|
| Approve | Completes run, saves output |
| Revise Goal | Opens inline editor for goal text |
| Stop | Terminates run |
| Retry | Restarts from beginning |
| View Output | Opens output in scratchpad |

### Keyboard Chords

| Action | Chord |
|--------|-------|
| Approve | `SPC t a` |
| Stop | `SPC t x` |
| Retry | `SPC t r` |
