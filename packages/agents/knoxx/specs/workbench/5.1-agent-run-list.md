## Behavior

### Layout

```
┌─────────────────────────────────────────────────────────────────┐
│ Agents                                     [+ New Task]        │
├──────────────┬──────────────────────────────────────────────────┤
│ Active runs  │ Run Detail                                       │
│ ──────────  │ (see 5.2)                                        │
│ ● synthesis  │                                                  │
│   q4-summary │                                                  │
│   ⏸ paused   │                                                  │
│              │                                                  │
│ ● ingestion  │                                                  │
│   devel-docs │                                                  │
│   ✓ complete │                                                  │
│              │                                                  │
│ Recent       │                                                  │
│ ──────────  │                                                  │
│ ○ synthesis  │                                                  │
│   weekly     │                                                  │
│   ✓ 2h ago   │                                                  │
└──────────────┴──────────────────────────────────────────────────┘
```

### Run Status

| Status | Icon | Description |
|--------|------|-------------|
| Running | `●` (pulse) | Currently executing |
| Paused | `⏸` | Awaiting approval |
| Complete | `✓` | Finished successfully |
| Failed | `✗` | Ended with error |

### List Sections

- Active: Running and paused runs
- Recent: Last 10 completed runs

### Interaction

- Click run → shows detail (5.2)
- "+ New Task" → opens task composer
- Hover → shows elapsed time
