## Behavior

### Layout

```
┌─────────────────────────────────────────────────────────────────┐
│ Focal: GraphWeaver                          [← Back] [Expand +]│
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  ┌────────────────────────────────────────────────────────────┐ │
│  │                                                            │ │
│  │    [OpenPlanner] ──edge: writes── [GraphWeaver]            │ │
│  │         │                              │                   │ │
│  │    edge: populates              edge: syncs               │ │
│  │         │                              │                   │ │
│  │    [KnowledgeLake]              [KnoxxGraph]              │ │
│  │                                                            │ │
│  └────────────────────────────────────────────────────────────┘ │
│                                                                 │
│  Node: GraphWeaver                                              │
│  strength:        ████░ (0.74)                                  │
│  trail heat:      ██░░░ recently cool                           │
│  status:          ⚠ contested — see 2 conflicting nodes         │
│  edge views:      [raw] [discovery] [structural] [evidence]     │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### Node Detail

- Label and type
- Strength indicator (visual bar)
- Trail heat (activity level)
- Status badges from signal vocabulary
- Edge view selector

### Neighborhood Expansion

- Default: 1 hop
- Expand button: 2 hops
- Click neighbor → becomes new focal node

### Edge View Types

| View | Shows |
|------|-------|
| raw | All edges without filtering |
| discovery | Edges that led to new knowledge |
| structural | Parent/child relationships |
| evidence | Provenance/supporting edges |
