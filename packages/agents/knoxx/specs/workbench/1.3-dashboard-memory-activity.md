## Behavior

### Activity Display

```
┌─────────────────────────────────────────────────┐
│ Recent Memory Activity                          │
├─────────────────────────────────────────────────┤
│ [node: GraphWeaver] reinforced ×12 today       │
│ [node: Tenant isolation] contested — 2 sources │
│ [node: SSO runbook §3.2] decaying — 8 days old │
└─────────────────────────────────────────────────┘
```

### Signal Types

| Signal | Display |
|--------|---------|
| High co-activation | "reinforced ×N today" |
| Conflicting evidence | "contested — N sources" |
| Low recent access | "decaying — N days old" |
| Derived from corrections | "shaped by N corrections" |
| Bridge edge | "cross-domain link to [topic]" |

All signals use Memory Signal Vocabulary chips (see 7.1).

### Interaction

- Click any line → navigates to Memory Inspector with that node as focal

---

## Exit Criteria

- [ ] Shows last 5 memory signals
- [ ] Uses Memory Signal Vocabulary chips (7.1)

- [ ] Empty state: "No recent memory activity"
