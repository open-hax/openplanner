### Publish Menu Options

When in `draft`:
- "Submit for Review" вҶ’ sets visibility to `review`
- Direct publish is disabled with tooltip

When in `review`:
- "Publish" вҶ’ sets visibility to `published`
- "Send Back to Draft" вҶ’ sets visibility to `draft`

When `published`:
- "Archive" вҶ’ sets visibility to `archived`
- "Edit" вҶ’ warns this will unpublish

### Confirmation Dialog

Before any state change:
- Show summary of change
- Require explicit confirmation
- Show consequences (e.g., "This will make the document public")

### Keyboard Chord

| Action | Chord |
|--------|-------|
| Open publish menu | `SPC p` |

## Acceptance Tests

1. Create document вҶ’ status is "draft"
2. Try to publish from draft вҶ’ option disabled
3. Submit for review вҶ’ status is "review"
4. Publish from review вҶ’ confirmation appears
5. Confirm publish вҶ’ status is "published"
6. Edit published document вҶ’ warning about unpublishing
