## Purpose

Show list of recent/active agent runs with status icons and quick actions.

---

## Scope

Create the agent run summary component that shows what agents are doing.

---

## Files to Create

| File | Purpose |
|------|---------|
| `src/components/dashboard/AgentRunSummary.tsx` | Agent run list component |

---

## Behavior

### Run List Display

```
┌─────────────────────────────────────────────────┐
│ Recent Agent Runs                    [View All] │
├─────────────────────────────────────────────────┤
│ ● Ingestion: devel-docs      ✓ done    3m ago  │
│ ● Synthesis: Q4 summary      ⏸ paused  5m ago  │
│ ● MT pipeline: batch 7       ✗ failed  12m ago │
└─────────────────────────────────────────────────┘
```

### Status Icons

| Status | Icon | Color |
|--------|------|-------|
| Running | `●` (animated pulse) | green |
| Paused | `⏸` | amber |
| Done | `✓` | green |
| Failed | `✗` | red |

### Quick Actions

- Paused runs: Show "Approve" button inline
- Clicking any row → navigates to Agent Workspace with that run selected
- "View All" link → navigates to Agent Workspace

### Data Source

Query agent run history from agent runtime (P1B dependency).

---

## Exit Criteria

- [ ] Shows last 5 runs (or fewer if not enough history)
- [ ] Status icons match run state
- [ ] Elapsed time shown relative (e.g., "3m ago")
- [ ] Paused runs show inline "Approve" button
- [ ] Clicking row navigates to Agent Workspace
- [ ] "View All" link navigates to Agent Workspace
- [ ] Empty state: "No recent agent runs"

---

## Out of Scope

- Full run details (see Agent Workspace)
- Starting new runs
- Run configuration

---

## Acceptance Tests

1. Load with active run → shows pulsing green icon
2. Load with paused run → shows approve button
3. Click approve → run resumes, button disappears
4. Click row → navigates to Agent Workspace with run selected
5. Load with no runs → shows empty state
