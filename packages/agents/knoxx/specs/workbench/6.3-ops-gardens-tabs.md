## Behavior

### Tab Layout

```
┌─────────────────────────────────────────────────────────────────┐
│ Ops                                                             │
│ [Events] [Dependency Garden] [Truth Garden]                     │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  (Events table / Dependency Garden / Truth Garden)              │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### Tabs

| Tab | Content |
|-----|---------|
| Events | Ops event table (6.1) |
| Dependency Garden | Workspace dependency topology |
| Truth Garden | Truth/control-plane state |

### Integration

- Gardens share status bar
- Gardens share keyboard system
- Gardens accessible from Ops only (not top-level nav)

### Dependency Garden

Shows workspace package dependencies as a graph or tree.
- Filter by package name
- See dependency chain
- Identify circular deps

### Truth Garden

Shows control-plane state and truth values.
- Current system state
- Recent state changes
- Audit trail
