# ChordOverlay Integration

**Parent**: `knowledge-ops-workbench-ux-breakdown.md`
**Epic**: 0. Shell Foundation
**Points**: 2
**Dependencies**: 0.1 Shell Layout Component

---

## Purpose

Spacemacs-style keyboard chord discovery overlay that reveals available actions when SPC is pressed.

---

## Scope

Integrate `@open-hax/uxx` ChordOverlay component into the shell and register workbench actions.

---

## Files to Create

| File | Purpose |
|------|---------|
| `src/shell/ChordProvider.tsx` | Chord context provider wrapping app |
| `src/shell/chord-actions.ts` | Action registry for workbench commands |

---

## Behavior

### Activation

- Press `SPC` (spacebar) → overlay appears
- Overlay shows all available actions grouped by category
- Press escape → overlay dismisses
- Press chord sequence → action executes, overlay dismisses

### Chord Registry

| Action | Chord | Category |
|--------|-------|----------|
| Dashboard | `SPC d` | Navigation |
| Content | `SPC c` | Navigation |
| Review | `SPC r` | Navigation |
| Memory | `SPC m` | Navigation |
| Agents | `SPC a` | Navigation |
| Ops | `SPC o` | Navigation |
| Search | `/` | Global |
| Approve item | `SPC a` | Review |
| Reject item | `SPC r` | Review |
| Skip item | `SPC s` | Review |
| Accept AI suggestion | `SPC i a` | Editor |
| Dismiss AI suggestion | `SPC i d` | Editor |
| New agent task | `SPC t n` | Agents |
| Stop agent run | `SPC t x` | Agents |

Note: Context-sensitive actions only appear when that context is active.

## Exit Criteria

- [ ] Pressing SPC reveals ChordOverlay
- [ ] Overlay shows all global actions
- [ ] Overlay shows context-specific actions
- [ ] Chord sequence executes correct action
- [ ] Overlay dismisses on Escape
- [ ] Overlay dismisses after action execution
- [ ] Works from any view
- [ ] Shows mode indicator in status bar

## Out of Scope

- Custom chord configuration UI
- Chord macro recording
