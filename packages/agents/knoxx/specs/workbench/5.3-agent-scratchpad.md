## Behavior

### Layout

```
┌─────────────────────────────────────────────────────────────────┐
│ Scratchpad                                                      │
├─────────────────────────────────────────────────────────────────┤
│ Sources                                                         │
│ ┌─────────────────────────────────────────────────────────────┐ │
│ │ [q4-kpis.md]         → from Chat                            │ │
│ │ [ops-report.md]      → from Search                          │ │
│ │ [agent output]       → from Run                             │ │
│ │ [+ Add from collection...]                                  │ │
│ └─────────────────────────────────────────────────────────────┘ │
│                                                                 │
│ Synthesis Prompt                                                │
│ ┌─────────────────────────────────────────────────────────────┐ │
│ │ Write a summary of Q4 performance using the provided docs   │ │
│ └─────────────────────────────────────────────────────────────┘ │
│                                                                 │
│ Output                                                          │
│ ┌─────────────────────────────────────────────────────────────┐ │
│ │ # Q4 Performance Summary                                    │ │
│ │                                                             │ │
│ │ Key metrics showed strong growth...                         │ │
│ │                                                             │ │
│ └─────────────────────────────────────────────────────────────┘ │
│                                                                 │
│ [Generate]  [Regenerate]  [Edit Output]                         │
│ [Copy]  [Export Markdown]  [Save to CMS]                        │
└─────────────────────────────────────────────────────────────────┘
```

### Source Assembly

- Add from: Chat answers, search results, agent outputs
- Remove sources with X button
- Drag to reorder

### Generation

- Generate: Calls synthesis with sources + prompt
- Regenerate: Re-runs with same inputs
- Edit Output: Opens output for manual editing

### Export Options

| Option | Effect |
|--------|--------|
| Copy | Copies to clipboard |
| Export Markdown | Downloads .md file |
| Save to CMS | Creates new document in CMS |

## Exit Criteria

- [ ] Source assembly panel accepts multiple sources
