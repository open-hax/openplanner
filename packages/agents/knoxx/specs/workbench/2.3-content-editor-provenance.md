## Behavior

### Panel Layout

```
┌─────────────────────────┐
│ PROVENANCE              │
│                         │
│ Sources used:           │
│ • knowledge-ops-wb.md   │
│ • chat: "3-pane layout" │
│ • correction by: riatz  │
│                         │
│ ─────────────────────── │
│                         │
│ MEMORY                  │
│                         │
│ node: Workbench UX      │
│ strength: ████░ medium  │
│ last reinforced: 2h ago │
│ co-activated with:      │
│ Agent Runtime, UXX      │
│                         │
│ [Expand in Memory →]    │
└─────────────────────────┘
```

### Provenance Data

| Section | Content |
|---------|---------|
| Sources | List of files/chats/answers used |
| Agent | Which agent contributed (if any) |
| Corrections | Who made corrections, when |

### Memory Node Summary

Shows the memory node associated with this document topic:
- Node name
- Strength indicator
- Last reinforced timestamp
- Co-activated concepts

## Exit Criteria
