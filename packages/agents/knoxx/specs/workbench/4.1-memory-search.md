## Behavior

### Layout

```
┌─────────────────────────────────────────────────────────────────┐
│ Memory  [🔍 Search nodes, edges, trails...]            [filters]│
├─────────────────────────────────────────────────────────────────┤
│  Results (12 found)                                             │
│  ────────────────                                              │
│                                                                 │
│  [GraphWeaver] [recently reinforced] [high-traffic]            │
│  strength: 0.74  |  last reinforced: 2h ago                     │
│                                                                 │
│  [OpenPlanner] [shaped by corrections]                          │
│  strength: 0.89  |  last reinforced: 1d ago                     │
│                                                                 │
│  [Knoxx Runtime] [decaying]                                     │
│  strength: 0.45  |  last reinforced: 8d ago                     │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### Search Scope

- Node labels and content
- Edge types
- Trail names
- Source provenance

### Result Display

Each result shows:
- Node label
- Memory signal chips (7.1)
- Strength indicator
- Last reinforced timestamp

### Filters

- Signal type (contested, decaying, etc.)
- Strength range
- Date range
- Node type

### Initial State

- Search bar focused by default
- No results shown until query entered
- Empty state: "Search to explore the knowledge graph"

---

## Exit Criteria

- [ ] Search bar focused by default
- [ ] Searches nodes, edges, and trails
- [ ] Results show node label + signal chips
- [ ] Results show strength and timestamp
- [ ] Filters available
- [ ] Click result → focal node view (4.2)
- [ ] Empty state before search
- [ ] No results state with suggestions

---

## Out of Scope

- Focal node detail (see 4.2)
- Graph visualization
- Historical view
