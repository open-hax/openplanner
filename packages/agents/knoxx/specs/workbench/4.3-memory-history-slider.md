## Behavior

### Slider Layout

```
┌─────────────────────────────────────────────────────────────────┐
│ History  [▶ replay]  ─────2026-04-09──────●──2026-04-11──      │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  Node state at: 2026-04-10 14:32                                │
│                                                                 │
│  strength:        ███░░░ (0.58)  ← was lower                    │
│  trail heat:      ████░ recently warm                           │
│  status:          ✓ stable                                      │
│                                                                 │
│  Events since:                                                  │
│  • Correction by riatz (reinforced +0.16)                      │
│  • Contested signal from conflicting source                     │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### Controls

- Slider: Drag to scrub through time
- Play button: Animate forward from selected point
- Pause button: Stop animation
- Date display: Shows currently selected timestamp

### Historical State

When viewing past state:
- Strength shows historical value
- Trail heat shows historical activity
- Status shows historical badges
- Events list shows what changed since

### Integration

- Appears below node detail in focal view
- Default position: current time (rightmost)
- Persists across node selections
