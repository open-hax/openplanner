**Parent**: `knowledge-ops-workbench-ux-breakdown.md`

**Dependencies**: None

## Purpose

Three-pane responsive shell with Context Bar (left), Main Canvas (center), and Inspection Panel (right). The Shell provides the internal layout for the workbench workplace, which is ONE workplace among many in the larger Knoxx app.

---

## Clarification (2026-04-12)

**The workbench is NOT a replacement for the main app.** It is one workplace among many (Chat, CMS, Ingestion, Query, Gardens, Runs, Translations, Admin, Workbench).

The main app navbar (visible in `app-shell__header`) provides top-level navigation between workplaces. The Shell provides internal navigation within the workbench workplace.

The "Header" shown in the original layout structure below was intended for a standalone workbench scenario. **This header is NOT implemented** because the main app navbar serves the top-level navigation purpose. The Shell focuses on the three-pane internal structure.

---

## Scope

Create the root layout component that establishes the three-pane grid structure. This is the foundation all workbench views render into.

## Behavior

### Layout Structure (Updated)

The Shell is nested inside `app-shell__main`, which is inside `app-shell` with the main navbar at the top:

```
┌──────────────────────────────────────────────────────────────────┐
│ MAIN APP NAVBAR (Chat, CMS, Ingestion, ..., Workbench, Admin)   │
├──────────────────────────────────────────────────────────────────┤
│ ┌────────────────────────────────────────────────────────────┐   │
│ │ Context  │ Main Canvas                           │ Inspect │   │
│ │ Bar      │ (swaps by view)                       │ Panel   │   │
│ │ (left)   │                                       │ (right) │   │
│ ├──────────┴───────────────────────────────────────┴─────────┤   │
│ │ Status Bar: collection, model, tokens, agent, mode         │   │
│ └────────────────────────────────────────────────────────────┘   │
└──────────────────────────────────────────────────────────────────┘
```

### Responsive Breakpoints

| Viewport | Context Bar | Inspection Panel |
|----------|-------------|------------------|
| < 768px | Icon strip only (collapsed) | Hidden (icon trigger) |
| 768px - 1024px | Icon strip + labels on hover | Drawer (slide out) |
| >= 1024px | Full width (200px) | Always visible (280px) |
