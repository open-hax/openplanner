## Behavior

### Layout

```
┌─────────────────────────────────────────────────────────────────┐
│ Ops                                  [filter ▾] [date range ▾] │
├─────────────────────────────────────────────────────────────────┤
│ Time         Type         Status   Summary                      │
│ ─────────── ──────────── ──────── ──────────────────────────── │
│ 10:42am     ingestion    ✓ done   devel-docs: 14 files, 2.1MB  │
│ 10:38am     embedding    ✓ done   14 chunks added              │
│ 10:31am     sync         ✓ done   OpenPlanner → GraphWeaver    │
│ 10:12am     policy check ⚠ warn   3 flagged segments (PII)     │
│ 09:55am     MT pipeline  ✗ error  batch7: timeout after 300s   │
│ 09:30am     ingestion    ✓ done   devel-code: 89 files, 4.2MB  │
└─────────────────────────────────────────────────────────────────┘
                                                [Inspect row →]
```

### Event Types

| Type | Description |
|------|-------------|
| ingestion | File/document ingestion |
| embedding | Vector embedding generation |
| sync | Graph/memory synchronization |
| policy check | PII/compliance scanning |
| MT pipeline | Machine translation batch |

### Status Icons

| Status | Icon | Color |
|--------|------|-------|
| Done | `✓` | green |
| Warning | `⚠` | amber |
| Error | `✗` | red |

### Filters

- Type dropdown: All / ingestion / embedding / sync / policy / MT
- Date range picker: last 1h / 24h / 7d / custom
