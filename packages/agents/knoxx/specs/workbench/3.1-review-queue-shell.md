## Behavior

### Layout

```
┌─────────────────────────────────────────────────────────────────┐
│ Review Queue                            [4 pending] [batch ▾]  │
├────────────────┬────────────────────────────────────────────────┤
│ Queue          │ Item Detail                                  │
│ ────────────  │ (see 3.2)                                    │
│ ● synthesis ✦  │                                              │
│   q4-summary   │                                              │
│                │                                              │
│ ● MT segment   │                                              │
│   batch7 #203  │                                              │
│                │                                              │
│ ● ingestion    │                                              │
│   devel-docs   │                                              │
│   4 flagged    │                                              │
└────────────────┴────────────────────────────────────────────────┘
```

### Queue Item Types

| Type | Badge | Source |
|------|-------|--------|
| Synthesis | `synthesis` | Agent-generated content |
| MT segment | `MT` | Machine translation |
| Ingestion | `ingestion` | Flagged during ingest |

### Queue Ordering

- Sort by confidence (lowest first)
- Secondary sort by age (oldest first)
- Paused items at top

### Batch Actions

- Approve all visible
- Export selected to scratchpad
