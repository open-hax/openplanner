### Content by Type

**Ingestion events:**
- Source path, file count, file types
- Chunks created, embeddings generated
- Processing duration

**Embedding events:**
- Model used, dimensions
- Chunks processed, tokens used
- Duration

**Sync events:**
- Source and destination
- Records synced, conflicts
- Duration

**Policy events:**
- Scan scope, rules applied
- Flagged items, severity
- Recommended actions

**MT events:**
- Source/target languages
- Segment count, success rate
- Errors (if any)

## Exit Criteria

- [ ] Shows full inputs
- [ ] Shows full outputs
- [ ] Shows duration
- [ ] Shows step trace
- [ ] Shows error trace if failed
- [ ] Links to related review item (if any)
- [ ] Click to collapse

---

## Out of Scope

- Editing events
- Re-running events
- Exporting traces

---

## Acceptance Tests

1. Click event row → detail expands
2. See inputs → all input fields shown
3. See outputs → all output fields shown
4. Failed event → error trace visible
5. Event with review item → link shown
6. Click expanded row → collapses
