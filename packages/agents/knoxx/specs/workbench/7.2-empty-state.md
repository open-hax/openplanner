## Behavior

### Component API

```tsx
interface EmptyStateProps {
  title: string;
  message: string;
  actionLabel?: string;
  onAction?: () => void;
  icon?: React.ReactNode;
}
```

### Layout

```
┌─────────────────────────────────────┐
│                                     │
│            [icon]                   │
│                                     │
│      All caught up!                 │
│                                     │
│   No items need your attention.     │
│   When documents are flagged for    │
│   review, they'll appear here.      │
│                                     │
│      [Start Writing]                │
│                                     │
└─────────────────────────────────────┘
```

### Usage Examples

**Dashboard attention cards:**
- Title: "All caught up!"
- Message: "No items need your attention."
- Action: "View all content"

**Review queue:**
- Title: "Queue empty"
- Message: "All items have been reviewed."
- Action: None

**Memory search results:**
- Title: "No matches found"
- Message: "Try different search terms or filters."
- Action: "Clear filters"

**Agent runs:**
- Title: "No agent runs"
- Message: "Start a new task to see runs here."
- Action: "New Task"
